镜像拉取与运行

Docker Hub 基础

Docker Hub 是官方的镜像存储和分发中心，提供公共和私有镜像仓库。

镜像命名规范

[registry]/[namespace]/[repository]:[tag]

示例：

nginx:latest
docker.io/library/ubuntu:20.04
registry.cn-hangzhou.aliyuncs.com/acs/nginx

镜像拉取命令

从 Docker Hub 拉取：

docker pull nginx
docker pull nginx:1.21
docker pull ubuntu:20.04

从阿里云镜像仓库拉取：

docker pull registry.cn-hangzhou.aliyuncs.com/acs/nginx
docker pull registry.cn-beijing.aliyuncs.com/library/redis:6.2

从私有仓库拉取：

docker pull localhost:5000/my-app:v1.0
docker pull myregistry.com:8080/project/api

运行容器示例

基础运行：

docker run hello-world
docker run -d nginx
docker run -it ubuntu:20.04 bash

端口映射运行：

docker run -d -p 8080:80 nginx
docker run -d -p 3306:3306 -e MYSQL_ROOT_PASSWORD=123456 mysql:8.0

数据卷挂载：

docker run -d -v /host/data:/container/data nginx
docker run -d -v myvolume:/var/lib/mysql mysql:8.0
