镜像管理核心命令

镜像查看

docker images
docker images -q
docker images --format "table {{.Repository}}\t{{.Tag}}\t{{.Size}}"
查看本地镜像列表，显示仓库名、标签和大小

镜像操作

docker pull image_name:tag
docker push image_name:tag
docker rmi image_id
docker rmi -f image_id
拉取、推送、删除镜像，-f 选项可强制删除

镜像构建

docker build -t image_name:tag .
docker build --no-cache -t image_name:tag .
docker tag source_image target_image
构建镜像，--no-cache 可禁用缓存，tag 可为镜像添加新标签

镜像清理

docker image prune
docker image prune -a
docker image prune -a --filter "until=2021-01-01"
删除悬空镜像，-a 可删除所有未使用镜像，--filter 可按时间筛选

系统管理命令

docker info
docker version
docker system df
docker system prune
docker system prune -a
系统信息、版本、磁盘使用情况、清理未使用资源

实用技巧

  • 使用 docker images -q 获取镜像ID列表
  • 结合 grep 过滤镜像：docker images | grep nginx
  • 使用 xargs 批量删除：docker images -q | xargs docker rmi
