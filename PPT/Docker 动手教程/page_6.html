环境安装配置

macOS
Windows
Linux

系统要求

  • macOS 11.0+ (Big Sur 或更高版本)
  • 4GB+ RAM (推荐 8GB+)
  • 4GB+ 磁盘空间

安装步骤

方法1: 官方安装包
  1. 下载 Docker Desktop for Mac
  2. 拖拽到 Applications 文件夹
  3. 启动应用程序
方法2: Homebrew
brew install --cask docker

安装验证

检查版本
docker --version docker-compose --version
运行测试容器
docker run hello-world
查看系统信息
docker info

实用提示

  • 配置镜像加速器以提升拉取速度
  • 增加资源分配（内存/CPU）以提高性能
  • 重启后检查 Docker 服务是否自动启动

系统要求

  • Windows 10 64-bit Pro/Enterprise/Education (版本 1903+) 或 Windows 11
  • 启用 Hyper-V 和 WSL 2
  • 4GB+ RAM (推荐 8GB+)

安装步骤

启用必要功能
  1. 开启 WSL 2 功能
  2. 安装 Linux 内核更新包
  3. 设置 WSL 2 为默认版本
安装 Docker Desktop
  1. 下载 Docker Desktop for Windows
  2. 运行安装程序
  3. 启用 WSL 2 后端集成

安装验证

检查版本
docker --version docker-compose --version
运行测试容器
docker run hello-world
查看系统信息
docker info

常见问题

  • WSL 2 安装失败 - 确保 Windows 更新到最新版本
  • 虚拟化未启用 - 在 BIOS 中启用 VT-x/AMD-v
  • 网络连接问题 - 检查防火墙设置

系统要求

  • 内核版本 3.10+ (推荐 4.0+)
  • 支持 overlay2 存储驱动
  • 64位架构 (x86_64/amd64)

Ubuntu/Debian 安装步骤

添加 Docker 官方仓库
curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg
安装 Docker Engine
sudo apt-get update sudo apt-get install docker-ce docker-ce-cli containerd.io docker-compose-plugin
启动服务
sudo systemctl start docker sudo systemctl enable docker
添加用户到 docker 组
sudo usermod -aG docker $USER

安装验证

检查版本
docker --version docker-compose --version
运行测试容器
docker run hello-world
查看系统信息
docker info

其他 Linux 发行版

CentOS, Fedora, RHEL 等其他发行版的安装步骤略有不同，请参考官方文档获取具体说明。

服务器环境提示

  • 生产环境建议使用 Docker Engine 而非 Desktop
  • 考虑配置远程访问以便远程管理
  • 设置日志轮转避免日志文件过大
